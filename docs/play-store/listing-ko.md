## 자세한 설명 (4000자 이내)

```
⚙️ Calibration Calculator — 현장 계측·제어 엔지니어를 위한 캘리브레이션 계산기

오프라인에서 동작하는 두 가지 핵심 계산 도구를 하나의 앱에 담았습니다.

▌루프 계산기 (Loop Calculator)
공정 변수(PV)를 4-20mA 출력 신호로 변환합니다.
• 입력·출력 범위 자유 설정
• 0% · 25% · 50% · 75% · 100% 퀵 체크 버튼
• 실시간 막대 그래프 및 백분율 표시
• 소수점 정밀 입력 — 장갑 끼고도 누르기 좋은 큰 키패드
• 온도·압력·유량·레벨 트랜스미터 검증에 즉시 활용

▌포지셔너 계산기 (Positioner Calculator)
밸브 포지셔너 캘리브레이션 시 DCS 설정값을 정확히 계산합니다.
• 현장 측정 Zero / Span mA 입력
• 원하는 DCS 표시 범위(예: -0.5% ~ 99.5%) 설정
• Zero Zone(-5% ~ 5%), Span Zone(95% ~ 105%) 2-Zone 시각화
• DCS 0% / 100% mA 결과를 즉시 산출 — 스프레드시트 불필요

▌현장 친화 설계
• 100% 오프라인 — 플랜트, 변전소, 옥외 현장에서도 동작
• 다크 테마 고대비 UI — 조명 환경에 강함
• 갤럭시 S24·S25에 최적화된 단일 화면 레이아웃
• 모든 키 입력에 햅틱 피드백 — 장갑 착용 시 입력 확인 용이
• 세로 화면 고정 — 멀티미터 잡고 흔들려도 회전 안 됨
• 무료 (배너 광고 1개)

▌사용 대상
공정 제어 엔지니어 · 계측 기술자 · DCS 커미셔닝 팀
플랜트 운영 인력 · MEP 엔지니어 · 산업 자동화 전문가
Yokogawa · Siemens · Honeywell · ABB · Emerson DCS 시스템 운용 인력

▌포함 기능
• 완전 오프라인 계산 엔진
• 마지막 입력 범위·값 자동 저장 (재실행 시 복원)
• 계정·로그인·클라우드 동기화 없음 — 입력값은 기기를 떠나지 않음
• Google AdMob 배너 광고
```
